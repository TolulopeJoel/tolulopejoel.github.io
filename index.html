Billboard Spaces

Chunkit

Chunkit offers a secure and efficient solution by splitting files into smaller, manageable chunks. This tool tackles challenges like slow transfers, limited storage, and unreliable connections. Chunkit intelligently splits files, optimizes transfers, and can seamlessly merge split files back.

Other projects.

Writer Watcher

Automatically tracks when favourite writers publish a post, eliminating the need for manual tracking, by monitoring specified blogs saving users time and effort.

Key Features:

  • Email Integration: It connects with Gmail to extract blog links from draft emails.

  • Automated Notifications: It sends email notifications whenever new content is published by the tracked writers.

  • Personalization: Users can track their favorite writers and receive updates tailored to their preferences.

Pixelgrab

A backend solution for a screen-recording chrome extension.

Key Features:

  • Video Stream: Establishes a secure connection to receive the video stream blobs captured by the Chrome extension.

  • Video Compression: Once a video is fully received, it is compressed to save space.

  • Thumbnail Extraction: A thumbnail image is extracted from the video for preview purposes.

Hackathon Host

An application for hosting hackathons, with functionalities such as enrolling for hackathons and making submissions based on hackathon submission type.

Key Features:

  • Create & Manage Hackathons: Authorized users can create hackathons with details, set submission type (image, file, or link), and verify user authorization.

  • User Registration & Submission: Users can register for hackathons and submit their work based on the specified type, with API validation for the submission format.

  • User Dashboard: Users can view their registered hackathons and submitted works.

Synkbrid

An application facilitating remote team productivity by offering robust capabilities for task management, deadline establishment, and seamless real-time collaboration.

Key Features:

  • Users can see changes made to tasks by others instantly, fostering a more synchronized workflow.

  • Tracking of tasks, including creation date, assigned personnel, deadlines, and progress updates

  • Calendar integration with tools for users to visualize timelines and track deadlines effectively.

  • Allows users to filter tasks based on their current status (e.g., To-Do, In Progress, Done) for better organization and focus.

Slime Road

A storefront for creators to sell their digital products directly to their audience. Creators can offer a variety of digital products (ebooks, software, courses, etc).

Key Features:

  • Ability for creators to easily upload, manage, and organize their digital products.

  • Support for fixed prices, pay-what-you-want, free, or subscription-based models.

  • Track and manage customer orders, including order history, download links, and fulfillment status.

  • Track sales performance through detailed reports.
